# 💰 Finance Dashboard

Aplicación web desarrollada con React para visualizar un resumen financiero personal.  
Incluye métricas clave, tarjetas informativas y gráfica comparativa de inversión vs deuda.

🔗 **Demo en producción:**  
https://finance-dashboard-eight-mu.vercel.app/

---

## 🚀 Tecnologías utilizadas

- React
- React Router DOM
- Recharts
- SCSS Modules
- Vite

---

## 📊 Funcionalidades

- Dashboard principal con resumen financiero
- Tarjetas con métricas clave:
  - Total invertido
  - Intereses generados
  - Créditos activos
  - Balance neto
- Gráfica comparativa (Inversión vs Deuda)
- Diseño responsive
- Arquitectura preparada para integración con API o estado global
